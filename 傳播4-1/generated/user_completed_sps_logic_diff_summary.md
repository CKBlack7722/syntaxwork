# 01-3 2026TCS問卷檢核.sps 與 問卷檢核複檢.sps 邏輯差異摘要

比對基準為 `2026年臺灣傳播調查資料庫正式調查問卷.docx` 中的跳題/需答/不需答/互斥說明，並以目前 `檢核程式套印-傳播4-1.xlsx` 的 `邏輯組` 作為結構化索引輔助檢查。以下只列實質邏輯差異；m/p 編號不同、寫法不同但功能相同者不列。

## 一、與問卷邏輯設定差異

### 1. 01-3：Q5 後續題 Q6-Q9 條件過窄

問卷原文：

- Q6: `Q5答(02)、(97)、(98)者，或K2答(90)者，不需答此題`
- Q7: `Q5答(02)、(97)、(98)者，或K3答(90)者，不需答此題`
- Q8: `Q5答(02)、(97)、(98)者，或K1答(90)者，不需答此題`
- Q9: `Q5答(02)、(97)、(98)者，或G7和G8和G9和G10和G11皆答(90)，不需答此題`

問卷邏輯應為：

- 應答：`not any(vQ5,2,97,98)` 且其他條件成立
- 不應答：`any(vQ5,2,97,98)` 或其他跳題條件成立

`01-3` 寫法使用：

- Q6-Q9 應答條件為 `vQ5=1`
- 不應答條件為 `vQ5~=1`

實質差異：若 Q5 答 3 或 4，依問卷不屬於跳題碼，後續題仍應依其他條件判斷是否需答；但 `01-3` 會把 Q5 非 1 視為應跳答。

`問卷檢核複檢.sps` 使用 `not any(vQ5,2,97,98)` / `any(vQ5,2,97,98)`，符合問卷。

### 2. 01-3：Q10 後續題 Q11-Q14 條件過窄

問卷原文：

- Q11: `Q10答(02)、(97)、(98)者，或K2答(90)者，或Q6答(01)者，不需答此題`
- Q12: `Q10答(02)、(97)、(98)者，或K3答(90)者，或Q7答(01)者，不需答此題`
- Q13: `Q10答(02)、(97)、(98)者，或K1答(90)者，或Q8答(01)者，不需答此題`
- Q14: `Q10答(02)、(97)、(98)者，或G7-G11皆答(90)，或Q9答(01)者，不需答此題`

問卷邏輯應為：

- 應答：`not any(vQ10,2,97,98)` 且其他條件成立
- 不應答：`any(vQ10,2,97,98)` 或其他跳題條件成立

`01-3` 寫法使用：

- Q11-Q14 應答條件為 `vQ10=1`
- 不應答條件為 `vQ10~=1`

實質差異：若 Q10 答 3 或 4，依問卷不屬於跳題碼，仍可能需答 Q11-Q14；但 `01-3` 會視為應跳答。

另外 `01-3` 的 Q13 不應答條件為：

```spss
(vQ10~=1|vK1m90=1&vQ8=1)
```

這和問卷的「Q10為2/97/98 或 K1=90 或 Q8=1」不同；且沒有括號時 `&` 和 `|` 的結合可能讓條件更窄。

`問卷檢核複檢.sps` 使用 `not any(vQ10,2,97,98)` / `any(vQ10,2,97,98)`，符合問卷。

### 3. 01-3：Q15 後續題 Q16-Q19 條件過窄

- Q16: `Q15答(02)、(97)、(98)者，或K2答(90)者，不需答此題`
- Q17: `Q15答(02)、(97)、(98)者，或K3答(90)者，不需答此題`
- Q18: `Q15答(02)、(97)、(98)者，或K1答(90)者，不需答此題`

- 不應答：`any(vQ15,2,97,98)` 或其他跳題條件成立

### 4. 01-3：缺少 CKZE1 的應答/不應答邏輯

問卷原文：

- `CKZE1：A09選(01)者，或E1選(02)、(97)、(98)，需回答此題`

問卷邏輯應包含：

- `vA9=1 | any(vE1,2,97,98)` 時，應答 `vCkZE1`
- `vA9~=1 & not any(vE1,2,97,98)` 時，不應答 `vCkZE1`

`01-3` 有 `vCkZE1` 的數值範圍檢核，但沒有找到對應的應答/不應答邏輯。它有 ZE2 的跳題邏輯，但 CKZE1 是另一個顯示/日期時間變項，不應只用 ZE2 取代。

`問卷檢核複檢.sps` 有：

```spss
do if (any(vA9,1) | any(vE1,2,97,98)) & any(1,vCkZE1_96).
do if (vA9~=1 & not any(vE1,2,97,98)) & any(0,vCkZE1_96).
```

符合問卷。

### 5. 01-3：Q22 的 Q20m02 條件寫法可疑

問卷原文：

- Q22：`Q20答(02)者，需回答此題`

`01-3` 寫法：

```spss
do if (vQ20m02=1)=1&any(1,vQ22m01_96 to vQ22m88_96).
```

這個 `(vQ20m02=1)=1` 不是一般建議寫法，可能造成 SPSS 語法或邏輯判讀問題。功能上應改為：

```spss
do if vQ20m02=1 & any(1,vQ22m01_96 to vQ22m88_96).
```

`問卷檢核複檢.sps` 使用 `any(vQ20m02,1)`，符合問卷。

### 6. 01-3：沒有資料檢核項目清單 m1701-m1747

`問卷檢核複檢.sps` 包含 `m1701-m1747` 的資料檢核項目清單，其中實際啟用 44 條，`1737-1739` 因條件空白未啟用。

`01-3 2026TCS問卷檢核.sps` 沒有 `m1701-m1747`。若這兩份都要作為最終完整檢核程式，則 `01-3` 缺少這一組人工補充檢核項目。

這一點不是 Word 問卷基本跳題邏輯，而是「資料檢核項目」補充邏輯差異。

## 二、兩個使用者彼此邏輯設定差異

### 1. Q5/Q10/Q15 後續題的判定方式不同

`01-3` 使用：

- Q5 後續題：`vQ5=1` / `vQ5~=1`
- Q10 後續題：`vQ10=1` / `vQ10~=1`
- Q15 後續題：`vQ15=1` / `vQ15~=1`

`問卷檢核複檢.sps` 使用：

- `not any(vQ5,2,97,98)` / `any(vQ5,2,97,98)`
- `not any(vQ10,2,97,98)` / `any(vQ10,2,97,98)`
- `not any(vQ15,2,97,98)` / `any(vQ15,2,97,98)`

這是實質邏輯差異，不是語法差異。

### 2. CKZE1 是否納入跳題邏輯不同

`問卷檢核複檢.sps` 有 CKZE1 應答/不應答邏輯；`01-3` 沒有找到對應邏輯。

### 3. Q22 條件寫法不同

`01-3` 使用 `(vQ20m02=1)=1`，`問卷檢核複檢.sps` 使用 `any(vQ20m02,1)`。

### 4. 資料檢核項目清單是否存在不同

`問卷檢核複檢.sps` 有 `m1701-m1747`；`01-3` 沒有此區段。

## 三、目前看起來功能相同或可視為相同者

以下項目雖然兩份寫法不同，但目前看起來功能方向一致，不列為實質差異：

- E5/E8/E11 題組：兩段需答條件用 AND，反向用 OR，兩份大致一致。
- H7_1：`01-3` 用 `range(vH1,1,991)&range(vH5,2,5)`，`複檢` 用排除 `0/997/998/9996` 與 `1/96/97/98`；以目前有效值範圍看，功能接近。
- Q20 後段主要題組 Q24/Q25/Q21/Q22/Q23/Q26/Q26A：除 Q22 的 `(vQ20m02=1)=1` 寫法外，主要應答/不應答方向可對應。
